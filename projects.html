Polygon NFT Marketplace

This user-facing project lets anyone mint NFTs with just a few clicks. It features respective filtering categories for items in collection, items created, all items available in the marketplace.

*NOTE* It may take a few moments for the Metamask prompt. You'll also need to have the Mumbai testnet configured on your Metamask to buy and mint NFTs (instructions in readme).

DEX Backend

A fully functional decentralised exchange backend built with SafeMath and unit tested against multiple ERC20 token mocks. I implemented the Bubble Sort algorithm to the exchanges Limit Order function to arrange prices according to the position taken (buy/sell).

SafeTX Wallet

The SafeTX smart contract is my attempt at programming as many restrictions as possible into the withdrawal function to ensure the receieving address matches an address registered by the wallet owner.

Enderun MultiSig

This boilerplate covers all the basic smart contract functionality you need to create a multisignature wallet app for your business or organisation. I created this contract as a proposal for my school's student government, but once naming is tweaked everything should work fine.
